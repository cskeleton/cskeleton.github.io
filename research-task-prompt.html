🔗 核心分析框架
地缘/政策 → 利率预期 → 美元/人民币/油金 → 全球风险资产 → 海外市场 → 港股 → A股映射

♻️ ReAct 查证循环格式
Hypothesis: 待验证的核心假设
Reason: 为什么值得查证，影响哪条主线
Action: 本轮检索/比对/交叉验证动作
Sub-agent: Yes/No，若Yes指派谁、补什么证据
Observation: 新增事实、数据、价格行为
Evidence For: 支持假设的证据列表
Evidence Against: 削弱/反驳假设的证据列表
Confidence: 高/中/低 或 0-100分
Next Step: 继续查什么 / 证据充分则结束
Sources: 来源名 | 发布时间 | URL
📋 最终报告必须包含
  • 任务执行状态 🤖 — 轮次、Agent启用、关键词、数据路径
  • 基础行情复盘 📊 — A股/港股/美股/大宗/汇率，带来源标注
  • 宏观事件穿透 🔍 — 72h~7天事件时间线，多语种预期差分析
  • 情绪仓位调制 🧠 — risk-on/off、拥挤度、四种情绪状态定性
  • ReAct查证摘要 ♻️ — 每轮假设、证据、保留/弱化/证伪逻辑
  • 三地市场预测 🔮 — 基准/乐观/风险情景 + 操作建议 + 失效条件
  • 黑天鹅与关键变量 ⚠️ — 3-5个观察变量 + 尾部风险提示
  • 溯源归档说明 🗂️ — 归档路径 + 核心来源清单
⚠️ 核心原则：禁止空泛判断。所有重要结论必须有至少两类证据交叉验证。不允许只写"情绪谨慎"而不给证据。所有判断必须可溯源、可审计。
